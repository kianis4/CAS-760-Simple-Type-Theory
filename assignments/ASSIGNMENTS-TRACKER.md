## üìÖ Assignment Strategy (ADHD-Friendly)

### When Assignment is Posted:

**Day 1:**
- [ ] Download assignment
- [ ] Read through completely
- [ ] Identify all questions/parts
- [ ] Estimate time needed
- [ ] Schedule work sessions

**Day 2-3:**
- [ ] Start with easiest problems first (build momentum)
- [ ] Work in 25-min blocks (Pomodoro)
- [ ] Note questions for office hours

**Middle:**
- [ ] Attend office hours if stuck
- [ ] Check discussion forum
- [ ] Continue working through problems

**2 Days Before Due:**
- [ ] All problems attempted
- [ ] Review and polish answers
- [ ] Check LaTeX compiles correctly

**1 Day Before Due:**
- [ ] Final review
- [ ] Compile final PDF
- [ ] Submit on Avenue

**Due Date:**
- [ ] Verify submission went through
- [ ] Breathe!
