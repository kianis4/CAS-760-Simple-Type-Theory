### 1Ô∏è‚É£ Inductive Sets (Slides 3-10)

**Generators** (Slide 4)
> Let $A$ be a nonempty set of values.
> A **generator** for $A$ is a value $g$ that satisfies one of:

| Type | Form | Description |
|------|------|-------------|
| Base | $g \in A$ | Constant element |
| Unary | $g : A \to A$ | Function from $A$ to $A$ |
| Binary | $g : A \times A \to A$ | Function from pairs to $A$ |
| n-ary | $g : A^n \to A$ | Function from n-tuples to $A$ |

**Closure Formula $X_g$** (Slide 5)
- For predicate $p : A \to \mathbb{B}$, define $X_g$ as:
  - If $g$ is base: $p(g)$
  - If $g : A \to A$: $\forall x : A. p(x) \Rightarrow p(g(x))$
  - And so on for n-ary generators

**Generated Set** (Slide 6)
$$gen(G) = \text{smallest set containing all base generators, closed under all function generators}$$

**Freely Generated** (Slide 7)
> $A$ is **freely generated** by $G$ if:
> 1. The generators are **exhaustive** (every element comes from some generator)
> 2. The generators are **disjoint** (different generators produce different elements)
> 3. The generators are **injective** (same generator with different inputs ‚Üí different outputs)

**Inductive Set Definition** (Slide 8)
> An **inductive set** is a pair $(A, G)$ where:
> - $A$ is a nonempty set
> - $G$ is a finite set of generators for $A$
> - $A$ is freely generated by $G$

#### Examples

**Boolean Values** (Slide 9)
$$\mathbb{B} = \{f, t\}$$
- Generators: $G = \{f, t\}$ (two base generators)
- $(\mathbb{B}, \{f, t\})$ is an inductive set

**Natural Numbers** (Slide 10)
$$\mathbb{N} = \{0, 1, 2, ...\}$$
- Generators: $G = \{0, S\}$ where $S : \mathbb{N} \to \mathbb{N}$ is successor
- $(\mathbb{N}, \{0, S\})$ is an inductive set
- **Induction principle:** To prove $P(n)$ for all $n$:
  - Base: Prove $P(0)$
  - Step: Prove $P(n) \Rightarrow P(S(n))$

### 2Ô∏è‚É£ Inductive Types (Slides 11-18)

**Constructors** (Slide 12)
> Let $a$ be a base type. A **constructor** for $a$ is one of:

| Kind | Form | Pure? | Recursive? |
|------|------|-------|------------|
| 1 | $c : a$ | ‚úÖ | ‚ùå |
| 2 | $c : \beta \to a$ (where $\beta \neq a$) | ‚ùå | ‚ùå |
| 3 | $c : a \to a$ | ‚úÖ | ‚úÖ |
| 4 | $c : \beta \to a \to a$ | ‚ùå | ‚úÖ |

**Pure vs Recursive** (Slide 13)
- **Pure:** Kind 1 or 3 (no external type $\beta$)
- **Recursive:** Kind 3 or 4 (takes $a$ as argument)

**Obligations** (Slides 14-15)
> Let $C$ be a finite set of constructors for $a$.
> The **obligations** $obl(C)$ are axioms ensuring:
> 1. **Exhaustiveness:** Every element comes from some constructor
> 2. **Disjointness:** Different constructors produce different elements
> 3. **Injectivity:** Same constructor, different inputs ‚Üí different outputs

**Inductive Type Definition** (Slide 16)
> Let $T = (L, \Gamma)$ be a theory where $L = (B, C)$, $a$ is a base type, and $\mathcal{C}$ is a finite set of constructors for $a$.
>
> $a$ is an **inductive type** in $T$ if:
> - All obligations $obl(\mathcal{C}) \subseteq \Gamma$

#### Example Theories

**Theory of Boolean Values (BOOLE)** (Slide 17)
```
Name: BOOLE
Language: L = ({o}, {T : o, F : o})
Constructors: {T, F} (both kind 1)
Axioms (obligations):
  1. T ‚â† F                    [Disjointness]
  2. ‚àÄx:o. x = T ‚à® x = F      [Exhaustiveness]
```

**Peano Arithmetic (PA)** (Slide 18)
```
Name: PA
Language: L = ({N}, {0 : N, S : N ‚Üí N})
Constructors: {0, S} (kinds 1 and 3)
Axioms (obligations):
  1. ‚àÄx:N. S(x) ‚â† 0                    [Disjointness]
  2. ‚àÄx,y:N. S(x) = S(y) ‚áí x = y       [Injectivity of S]
  3. Induction schema                   [Exhaustiveness]
```

### 3Ô∏è‚É£ Inductive Type Theory Extensions (Slides 19-24)

**Definition** (Slides 20-21)
> Let $T_1 = (L_1, \Gamma_1)$ and $T_2 = (L_2, \Gamma_2)$ be theories.
>
> $T_2$ is an **inductive type theory extension** of $T_1$ by $(a, C)$ if:
> - $L_2$ extends $L_1$ with base type $a$ and constructors $C$
> - $\Gamma_2 = \Gamma_1 \cup obl(C)$

**Key Theorem** (Slide 21)
> Let $T_2$ be an inductive type theory extension of $T_1$ by $(a, C)$. Then:
> 1. $T_2$ is a **conservative extension** of $T_1$
> 2. $a$ is an **inductive type** in $T_2$
> 3. Recursion and induction principles hold for $a$

**Extension Module Template** (Slide 22)
```
Inductive Type Theory Extension X.Y
  Name: NAME
  Extends: T‚ÇÅ
  New base type: a
  Constructors: C = {...}
```

**Example: BOOLE-ALT** (Slide 23)
```
Name: BOOLE-ALT
Extends: Empty theory
New base type: o
Constructors: {T : o, F : o}
```

**Example: PA-ALT** (Slide 24)
```
Name: PA-ALT
Extends: Empty theory
New base type: N
Constructors: {0 : N, S : N ‚Üí N}
```

## üîë Key Concepts

| Term | Definition |
|------|------------|
| Generator | Value that builds elements of a set (base or functional) |
| $gen(G)$ | Smallest set closed under generators $G$ |
| Freely generated | Exhaustive, disjoint, injective generators |
| Inductive set | $(A, G)$ where $A$ freely generated by $G$ |
| Constructor | Typed generator for inductive type |
| Pure constructor | No external type parameter |
| Recursive constructor | Takes the inductive type as argument |
| Obligations | Axioms ensuring well-formed inductive type |
| Induction principle | Proof method: base case + inductive step |
