## üìù Lecture Notes

### What is Simple Type Theory?

**Definition (from slides):**
- A classical version of higher-order logic extending first-order logic
- Church introduced practical form in 1940 (Church's type theory)
- Tailored for reasoning with functions
- Alonzo = version that admits undefined expressions

**Why STT?**
- Practical logic for expressing mathematical ideas
- Foundation for proof assistants and math software
- Handles partial functions and definite descriptions

## üîë Key Concepts

| Term | Definition |
|------|------------|
| Simple Type Theory | Classical higher-order logic extending FOL, introduced by Church (1940) |
| Church's Type Theory | The practical form of STT tailored for reasoning with functions |
| Alonzo | A version of STT that admits undefined expressions (partial functions) |
| Higher-Order Logic | Logic that allows quantification over predicates and functions |
| Predicate Logic | Logic with predicates (properties/relations) and quantifiers |
| Little Theories Method | Approach to organizing mathematical knowledge as networks of theories |
